# Library App - Backend

Backend REST API for a library management web application.

## Description
Library App is a web application that allows users to browse books, manage their personal shelf, leave reviews, and check out books.  
The backend is implemented using Java and Spring Boot and exposes REST APIs consumed by the frontend application.

## Tech Stack
- Java
- Spring Boot
- Spring Data JPA
- Spring Data REST
- Lombok
- REST APIs
- MySQL
- Maven
- Auth0 (Authentication & Authorization)
- Stripe (For payment processing)

## Features
- CRUD operations
- Authentication and authorization implemented using Auth0
- Role-based access control (basic)
- Secured REST endpoints
- Database integration with MySQL

## Architecture
- Layered architecture (Controller / Service / Repository)
- JPA entity relationships
- RESTful API design
